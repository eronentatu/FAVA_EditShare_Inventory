# EditShareLTO_BackUp_Search

Static search page generated from Excel workbook.

## How to host on GitHub Pages
1. Create a repo named `{repo_name}` (or any name you prefer).
2. Upload these files to the repo root.
3. Go to Settings → Pages, select `main` branch and `/ (root)` folder.
4. GitHub Pages will provide you a public link like:
   https://YOURUSERNAME.github.io/EditShareLTO_BackUp_Search/

Open `index.html` in your browser or share the link.
